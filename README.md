# 🛒 Blinkit Sales Analysis Dashboard

This project presents a Power BI dashboard that analyzes Blinkit sales data to generate meaningful business insights. It helps visualize sales performance, product distribution, and customer ratings using interactive charts and KPIs.

## 📊 Key Insights
- Total Sales Performance
- Average Sales per Item
- Product Category Analysis
- Outlet Performance
- Customer Rating Analysis

## 🛠 Tools Used
- Microsoft Power BI
- Data Visualization
- Business Intelligence

## 📁 File
BLINKIT SALE ANALYSIS.pbit – Power BI template file containing the dashboard.

## 🚀 How to Use
1. Download the `.pbit` file from this repository.
2. Open it in **Power BI Desktop**.
3. Load the dataset if prompted.
4. Explore the interactive dashboard.
